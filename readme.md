# Parallel-Systems-Threading

Αυτό το repository περιλαμβάνει τις υλοποιήσεις των τεσσάρων υποερωτημάτων. Κάθε αρχείο αντιστοιχεί σε ένα υποερώτημα.

## Περιεχόμενα

- `1.1` — **Μέθοδος Monte Carlo για υπολογισμό του π με Pthreads**
    - Υλοποίηση της μεθόδου Monte Carlo για εκτίμηση του π, τόσο σειριακά όσο και παραλληλοποιημένα με χρήση Pthreads. Περιλαμβάνει μέτρηση και σύγκριση των χρόνων εκτέλεσης.

- `1.2` — **Ανανέωση κοινόχρηστης μεταβλητής με νήματα**
    - Υλοποίηση με πολλαπλά νήματα να αυξάνουν μια κοινόχρηστη μεταβλητή. Δύο εκδοχές συγχρονισμού: (1) με mutex, (2) με ατομικές εντολές. Σύγκριση απόδοσης των δύο μεθόδων.

- `1.3` — **Παράλληλη ανανέωση κοινόχρηστου πίνακα**
    - Τροποποίηση της προηγούμενης υλοποίησης. Κάθε νήμα αυξάνει ξεχωριστό στοιχείο ενός κοινόχρηστου πίνακα. Εξετάζεται αν απαιτείται συγχρονισμός και πώς μεταβάλλεται η απόδοση με την αύξηση του αριθμού νημάτων.

- `1.4` — **Υλοποίηση δικών μας κλειδωμάτων ανάγνωσης-εγγραφής**
    - Υλοποίηση read-write locks με χρήση condition variables και mutex, με δύο στρατηγικές: προτεραιότητα σε αναγνώστες και προτεραιότητα σε εγγραφείς. Σύγκριση απόδοσης των δύο εκδοχών.

---

Κάθε πρόγραμμα συνοδεύεται από Makefile για εύκολη μεταγλώττιση. Αναλυτικές οδηγίες, αποτελέσματα και παρατηρήσεις υπάρχουν εντός των πηγαίων αρχείων ως σχόλια.
